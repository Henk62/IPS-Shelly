# IPS-ShellyEM
   Dieses Modul ermöglicht es, ein Shelly EM in IP-Symcon zu integrieren.
     
   ## Inhaltverzeichnis
   1. [Konfiguration](#1-konfiguration)
   2. [Funktionen](#2-funktionen)
   
   ## 1. Konfiguration
   
   Feld | Beschreibung
   ------------ | ----------------
   MQTT Topic | Hier wird das Topic (shellyem-deviceid) des Shelly EM eingetragen. Dazu muss zur Zeit die folgende URL aufgerufen werden: http://ShellyIP/settings dort ist derHostname zu finden. Der Hostname ist die DeviceID!
   
   ## 2. Funktionen
   
Zur Zeit sind keine öffentlichen Funktionen vorhanden.
